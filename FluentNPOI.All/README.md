# FluentNPOI.All

Complete FluentNPOI package bundle - install this to get all features.

## Installation

```bash
dotnet add package FluentNPOI.All
```

## Included Packages

| Package | Description |
|---------|-------------|
| FluentNPOI | Core Excel read/write functionality |
| FluentNPOI.Pdf | PDF export using QuestPDF |
| FluentNPOI.Streaming | Large file streaming with ExcelDataReader |
| FluentNPOI.Charts | Chart generation with ScottPlot |

## Usage

After installing, all FluentNPOI functionality is available:

```csharp
using FluentNPOI;
using FluentNPOI.Pdf;
using FluentNPOI.Streaming;
using FluentNPOI.Charts;

// Use any FluentNPOI feature
```
